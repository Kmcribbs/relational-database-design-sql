# Relational Database Design Using SQL

## Overview
This project demonstrates advanced database design techniques using SQL, including table relationships, constraints, indexing, and data integrity enforcement.

## Skills Used
- SQL (CREATE TABLE, ALTER TABLE, INSERT)
- Relational database design
- Foreign keys and constraints
- Indexing for performance optimization

## Key Features
- Created multiple related tables using foreign key relationships
- Implemented cascading updates and deletes
- Designed indexes to improve query performance
- Applied constraints to maintain data integrity

## File Included
- database_schema.sql: SQL script defining the relational database structure
